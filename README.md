dotfiles
====

## Description

Repository for configurations of below packages:

* git
* bash / zsh
* screen / tmux
* emacs

## Requirement

* zsh
  * [prezto](https://github.com/sorin-ionescu/prezto)

* tmux
  * 1.9~
  * tpm (will be installed in setup scripts)

## Install

```
$ ./setup.sh
```
